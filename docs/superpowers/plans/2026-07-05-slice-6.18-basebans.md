# Slice 6.18 — `basebans` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** Ship `sm_ban` / `sm_unban` / `sm_addban` with a persistent, connect-enforced SteamID ban list.

**Architecture:** A host-global `BAN_CACHE` in core (mirrors the admin cache), a `ClientConnect` SourceHook in the shim (sibling to the proven `ClientCommand` hook) that rejects banned SteamIDs, an engine-generic `@s2script/bans` core prelude (mirrors `@s2script/admin`) loading `bans.json`, and a CS2 `@s2script/basebans` plugin for the commands.

**Tech Stack:** Rust (core cdylib), C++ (Metamod shim + SourceHook), TypeScript (the plugin), embedded JS (the core prelude).

## Global Constraints

- **Core stays engine-generic.** A SteamID/ban map is Source2-generic (identical to the existing admin cache). No CS2 symbols in `core/`. `ISource2GameClients` is a Source2 *engine* interface → the connect hook lives in the shim only. Both `check-core-generic` boundary gates must stay green. Only `@s2script/basebans` (`plugins/`) is CS2.
- **Mirror existing patterns verbatim** — do not invent new shapes. The admin cache (`v8host.rs:353-359, 3370-3423`), the admin prelude (`v8host.rs:760-807`), the admin natives registration (`v8host.rs:2997-3001`), the `s2script_core_dispatch_client_command` ffi export (`ffi.rs:159-167`), and the `ClientCommand` SourceHook (`s2script_mm.cpp:72, 1152, 1557, 1646`) are the templates.
- **Degrade-never-crash.** Malformed `bans.json` → skip-with-warn, never throw. `ban_check` panic → return 0 (fail-open: a core bug must never wedge all connections).
- **No new `S2EngineOps` ABI op.** `ban_check` is a shim→core ffi export; `client_kick` already exists. The ABI ops struct is untouched.
- **`bans.json` auto-gen template must be VALID JSON** (a `//`-comment template fails `JSON.parse` — the admin-template live-gate fix).
- Commit messages end with `Claude-Session: https://claude.ai/code/session_01G8RQTQTdmeczdE6oihp8Pn` (no backticks in `-m`; use `-F -`). Sign with the on-disk ed25519 key (already configured).

### Core: the `ban_check` ffi export

- [ ] **Step 4 — internal `ban_check`.** In `v8host.rs`, add a pub fn:
  ```rust
  /// Returns Some(reason) if `xuid` is currently banned (perm or unexpired), else None.
  pub fn ban_check(xuid: u64, now: i64) -> Option<String> {
      let key = xuid.to_string();
      BAN_CACHE.with(|m| {
          m.borrow().get(&key).and_then(|(until, reason)| {
              if *until == 0 || *until > now { Some(reason.clone()) } else { None }
          })
      })
  }
  ```

- [ ] **Step 5 — the ffi export** in `ffi.rs` (mirror `s2script_core_dispatch_client_command` at `:159`, `catch_unwind`/panic→0):
  ```rust
  #[no_mangle]
  pub extern "C" fn s2script_core_ban_check(xuid: u64, now: i64, out_reason: *mut c_char, cap: c_int) -> c_int {
      std::panic::catch_unwind(|| {
          match v8host::ban_check(xuid, now) {
              Some(reason) => {
                  if !out_reason.is_null() && cap > 1 {
                      let bytes = reason.as_bytes();
                      let n = std::cmp::min(bytes.len(), (cap as usize) - 1);
                      unsafe {
                          std::ptr::copy_nonoverlapping(bytes.as_ptr(), out_reason as *mut u8, n);
                          *out_reason.add(n) = 0;
                      }
                  }
                  1
              }
              None => 0,
          }
      }).unwrap_or(0)
  }
  ```

  ```js
  function __s2_ban_parseFile(text) {
    var obj; try { obj = JSON.parse(text); } catch (e) { return; }
    if (!obj || typeof obj !== "object") return;
    for (var sid in obj) {
      if (sid === "_help" || !obj.hasOwnProperty(sid)) continue;
      var e = obj[sid];
      if (!e || typeof e !== "object") continue;
      var until = (typeof e.until === "number") ? e.until : 0;
      var reason = (typeof e.reason === "string") ? e.reason : "";
      __s2_ban_set(String(sid), until, reason);
    }
  }
  function __s2_ban_load() {
    var text = __s2_config_read_raw("bans");
    if (text === null || text === undefined) {
      __s2_config_write_raw("bans", '{\n  "_help": "SteamID64 -> { until: <unix seconds, 0 = permanent>, reason }. Managed by sm_ban/sm_unban."\n}\n');
      text = __s2_config_read_raw("bans");
      if (text === null || text === undefined) return;
    }
    __s2_ban_parseFile(text);
  }
  function __s2_ban_rewrite() {
    var list = JSON.parse(__s2_ban_list());
    var obj = {};
    for (var i = 0; i < list.length; i++) obj[list[i].steamid] = { until: list[i].until, reason: list[i].reason };
    __s2_config_write_raw("bans", JSON.stringify(obj, null, 2) + "\n");
  }
  var __s2_bans = {
    add: function (steamId, minutes, reason) {
      var until = (minutes > 0) ? (Math.floor(Date.now() / 1000) + Math.floor(minutes) * 60) : 0;
      __s2_ban_set(String(steamId), until, reason ? String(reason) : "");
      __s2_ban_rewrite();
    },
    remove: function (steamId) { var r = __s2_ban_remove(String(steamId)); __s2_ban_rewrite(); return r; },
    get: function (steamId) { var s = __s2_ban_get(String(steamId)); return s ? JSON.parse(s) : null; },
    list: function () { return JSON.parse(__s2_ban_list()); },
    reload: function () { __s2_ban_clear(); __s2_ban_load(); }
  };
  if (!__s2_ban_mark_loaded()) { __s2_ban_load(); }
  globalThis.__s2pkg_bans = { Bans: __s2_bans };
  ```
