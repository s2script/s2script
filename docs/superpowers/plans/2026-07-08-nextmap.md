# nextmap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Ship the SM `nextmap` plugin — a `maplist.txt` rotation with `sm_setnextmap` override, the `nextlevel` cvar (basetriggers' `nextmap` trigger reads it), and an auto-changelevel at `mp_maxrounds`/`mp_timelimit`.

**Architecture:** One CS2/game-layer plugin in `disabled/nextmap/` (opt-in). JS-only, no sniper. Composes `@s2script/server` (getCvar/setCvar/command/gameTime/isMapValid), `@s2script/events` (round_end), `@s2script/timers` (delay), `@s2script/commands`, `@s2script/admin`, `@s2script/chat`, `@s2script/frame`, `@s2script/config`.

**Tech Stack:** TypeScript (pure ESM), `s2script build`.

## Global Constraints

- Ships opt-in in `disabled/nextmap/` (output `disabled/nextmap/dist/_s2script_nextmap.s2sp`).
- Pure ESM named imports; the 5E.1 typecheck gate MUST pass.

- Plugins PERSIST across a changelevel — reset per-map state via a `Server.mapName` poll on `OnGameFrame` (the rockthevote/nominations pattern), NOT `onLoad`-per-map.

### Task 2: Map-end detection (round + time limits) + changeToNext

**Files:**
- Modify: `disabled/nextmap/src/plugin.ts`.

**Interfaces:**
- Consumes (from Task 1): all state + `rotationNext` + the `changeToNext` stub (replace its body) + `pollTick` (add the timelimit branch). Adds `Events.on("round_end", handler)` (`@s2script/events`), `delay` (`@s2script/timers`), `Server.command`/`gameTime`/`getCvar`, `config.getInt`/`getBool`.

- [ ] **Step 1: Fill `changeToNext()`.**
  ```
  function changeToNext(): void {
    if (changing) return;
    changing = true;
    const next = override ?? rotationNext(currentMap);
    if (!next) { console.log("[nextmap] no next map available"); return; }
    if (!/^[A-Za-z0-9_]+$/.test(next.name) || (next.workshopId !== null && !/^[0-9]+$/.test(next.workshopId))) {
      console.log("[nextmap] next map failed validation: " + JSON.stringify(next)); return;
    }
    const secs = config.getInt("nextmap_change_delay");
    Chat.toAll("[nextmap] Changing to " + next.name + " in " + secs + "s");
    delay(secs * 1000).then(() => {
      Server.command(next.workshopId ? "host_workshop_map " + next.workshopId : "changelevel " + next.name);
    }).catch(logErr);
  }
  ```

- [ ] **Step 2: `round_end` handler in `onLoad`.**
  ```
  Events.on("round_end", () => {
    if (changing) return;
    roundsPlayed++;
    const max = parseInt(Server.getCvar("mp_maxrounds"), 10);
    if (max > 0 && roundsPlayed >= max) changeToNext();
  });
  ```

- [ ] **Step 3: timelimit branch in `pollTick`** (after the map-change block, on the same-map path):
  ```
  if (!changing && config.getBool("nextmap_use_timelimit")) {
    const tl = parseInt(Server.getCvar("mp_timelimit"), 10);
    if (tl > 0 && Server.gameTime >= tl * 60) changeToNext();
  }
  ```
